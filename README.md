# 📚 Hibernate Book Management System

A simple Java console application built using **Hibernate (JPA)** for performing **CRUD operations** on a Book database.

This project demonstrates how Hibernate ORM interacts with a relational database using **EntityManager, EntityTransaction, and Persistence API**.

---

## 🚀 Features

- Add new books to database
- Find book by ID
- Update book details
- Delete book
- Menu-driven console interface
- Hibernate ORM integration

---

## 🛠 Technologies Used

- Java
- Hibernate ORM
- JPA (Jakarta Persistence)
- MySQL / Any relational database
- Maven (optional)
